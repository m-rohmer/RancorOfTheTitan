Prototype d'un jeu réalisé dans le cadre du cours de conception de jeux vidéo, automne 2017, UQAC.

Le jeu a été développé sous Unity, l'ensemble des fichiers nécessaires à son fonctionnement ou sa modification sont présents sur ce dépot.

Retrouvez de plus amples informations dans le PDF nommé rapport_final.pdf également joint dans ce dépot.
--------------------------------------

Contrôles du jeu :

  Dans ce jeu, le joueur va assister le personnage principal (l'héroïne Atalante) afin de l'aider à atteindre la fin du niveau.
Pour cela, il doit déjouer les pièges présents sur le chemin de la protagoniste et dispose pour ce faire de 3 pouvoirs :
  - l'Amphore, capable de faire grandir de manière instantanée les plants de vigne, les rendant ainsi possibles à grimper pour notre héroïne
  - la Pluie, afin de remplir les fosses dont le fond est tapissé de piques ; Atalante pourra alors nager à leur surface pour les traverser sans encombre
  - le Vent, effectuant une poussée verticale permettant à la jeune femme de sauter plus loin et ainsi passer au dessus de fosses trop profondes pour être franchies autrement
  
  Afin d'utiliser ces différents pouvoirs, le joueur devra faire preuve de rapidité et d'anticipation : il devra sélectionner à l'aide de sa souris le pouvoir qu'il souhaite utiliser en cliquant sur l'icône correspondant ; maintenir le clic en déplaçant son curseur de souris jusqu'à l'endroit choisi ; relâcher son clic pour déchaîner les effets du pouvoir.
